# DEOCSU
DEOCSU(**DE**ep-learning **O**ptimized **C**hIP-exo peak calling **SU**ite) is an deep-learning based ChIP-exo peak calling tool based on deep convolutional neural network model. The tool is named after the lab mascot pet DEOCSU[d…ôksu:] whose main staff is Seojoung Park in SBML-Kim lab.
